## 使用方法

```js
// 通过 npm 安装的方式
var ccplayer = require('cc-player');
ccplayer.play(...);

// 通过直接引用的方式会在 `window` 上添加一个变量 `CCPlayer`
// 直接使用该变量即可。
CCPlayer.play(...);
```
